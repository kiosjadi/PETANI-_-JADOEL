Petani Jadoel AI - Static Web Template (FREE)
Files included:
- index.html
- style.css
- app.js
- petani_jadoel.json
- images/placeholder.jpg

How to use (Netlify drag & drop):
1. Zip the folder 'petani_jadoel_web' or drag the folder content to https://app.netlify.com/drop
2. After upload, Netlify will give a live URL (e.g. yoursite.netlify.app).
3. To edit data, open 'petani_jadoel.json' and modify/add items.
4. To use real photos, replace images/placeholder.jpg and update JSON to include image filenames (and update app.js to use them).

Notes:
- This is a static demo (no server). For dynamic DB, we can add API + backend later.
- If you want, I can help deploy to Netlify or convert to dynamic version.
